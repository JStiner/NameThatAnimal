# Animal Name Generator

Single-file static animal name generator built to behave like a local HTML project.

## What it is

- One `index.html` file
- All CSS and JavaScript embedded
- Generates names locally in the browser
- No backend
- No build step
- Good fit for GitHub Pages

## Features

- Preferred First / Middle / Last inputs
- Name length selector: 1 to 5 words
- Style selector:
  - Nickname
  - Short Name
  - Full Name
  - Formal Name
  - Show / Stage Name
- Male / Female / Neutral name pool selector
- Animal selector
- One-name-at-a-time output
- Large generated-name display
- Copy-friendly text box
- Try Again button
- Built-in admin section for device-level animal list edits
- Reset to Default restores the built-in animal list in the HTML file

## Upload to GitHub

1. Create a new repository
2. Upload `index.html` and `README.md`
3. Commit
4. Enable GitHub Pages if desired

## Notes

- Device-level animal changes are stored in browser local storage
- Reset to Default clears local changes and reverts to the built-in list
