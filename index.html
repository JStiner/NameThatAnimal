How It Works

• The default animal list is built into this HTML file.
• Device-level edits are saved in your browser using local storage.
• Added animals appear in the dropdown immediately.
• Hidden animals are removed only on this device until restored or reset.
• Reset to Default clears local edits and reloads the built-in list.
